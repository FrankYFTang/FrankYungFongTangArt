# 卡媚爾的夕陽
Copyright (c) 2023, Frank Yung-Fong Tang


2023年12月20日
```
我躺在卡媚爾的沙灘上
守著同學們的鞋包和大衣
曬著金色橙紅的夕陽
看著那矅眼的火球微掛慢沈下疊層迎面的海浪
風冷
光暖
刺眼
我橫躺著
沙細
不濕
只怕它們鎖進襪中
看著海水旁戲鬧的人群
背光
我分不出誰是和我來的那群
單獨在後頭守著
沒和他們下去瘋

金色的陽光
由下巴向上照來
手机中的我
在這自然光中
膚色唯美起來
橙紅色系
似乎讓陰影看來微紫
面容中的我
好似時空交錯

鳥看的視角
目光不是沈思
而是凝望
緊閉的嘴角
不是剛毅
只是寒風中的等候

在幸福和孤單的臨界點遊移俳佪
只有祂能滿足我欠缺的渴望
多年以來我尋錯了對象
錯誤的期盼
不必要不可能會來的等待
何必強求
何必追尋
放下
去享受唯祂能滿足近在眼前的必然
```
