## 送子行
Dec 27, 2023 
```
聚散一年又匆匆
送君別離心懷愁
祝汝康健親神恩
願儂樂喜稚心守
```
## 傍溪古廊
Dec 26, 2023 
```
傍溪古廊紅燈艷
映水對樓岸燭點
幽遊閒情無人伴
孤身暗夜知誰眠
```

## 天堂鳥
Dec 23, 2023 
```
天堂鳥兒冬晨燦
昂首驕立不畏寒
紫羽皇袍眺蒼遠
艷頂枝展輝金冠
```
晨起携犬購飲，見天堂鳥花，成詩七言一首

## 金霞耀輝
Dec 17, 2023 
```
蒼海無際浪疊層
金霞耀輝夕彩紛
岸客戲水細沙軟
臥觀落日橙黃昏
```
昨㩦諸生前海邊賞夕，成詩一首

## 待君還
Dec 6, 2023 
```
枝枯暮色前
葉寂空對寒
湖灯初點靜
默待君夫還
```
