## 梗犬遊苑  
Jan 1, 2023

```
空苑無學子
頑遊梗犬一
不聞詩書讀
唯吠嬉鬧聲
```
週日下午與妻㩦犬博弟遊迪安薩校園。
## 日現盈凸月  
Jan 1, 2023

```
銀月盈凸蔚藍天
閒雲淡泊散輕眠
雀鳥三二悠遊過
枝枯葉落靜待春
```
下午藍天見月盈凸於天，在枯泊蹄諾市紀念公園成詩。
## 迎新年  
Jan 1, 2023

```
雀啼頌露清
喚我起早振
道濕朝陽好
莫負再來春
```
## 香檳雪景二  
Jan 1, 2023

```
白雪鋪地彩霞天
空椅對坐冰湖寒
憶昨白鵝鴛鴦嬉
今暮獨汝單影徊
```
見友胡意香檳雪景湖照有感
