## 鼻毛有
July 12, 2023 
```
頂禿缺毛鼻中有
長睫不見勁眉粗
美醜非由人定佐
上主鑒心知品俗
```

## 綠青球道
July 3, 2023 
```
綠樹綠枝綠蔭涼
青草清心青少郎
球場求賽啾聲近
道旁禱祈道康莊
```

## 「蒜泥雞胸肉片佐小黃瓜」
July 2, 2023 
```
穌胸淨白嫩滑軟
翡裙翠帶衣青歡
丹脂朱漿華清浴
入口芳芬勝玉環
```
