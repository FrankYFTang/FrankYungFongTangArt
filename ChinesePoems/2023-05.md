## 大都会
May 28, 2023 

```
萬畫齊陳大都会
千像仙展百柱林
今古藝匠一宮聚
縱橫西東顯瓏玲
```

## 牡丹歌
May 26, 2023 

```
百瓣含苞羞飛燕
渾圓粉透似玉環
不求明皇傾國寵
只惜詩仙露華濃
```

## 高樓頌
May 26, 2023 

```
萬樓爭高攀天寵
欲達青蒼把酒歡
銀塔成林巨佇遠
遥觀靜笑塵俗凡
```
## 巨樓頌
May 24, 2023 

```
巨方聳分天
暗影畫蒼尖
鏡牆日渲返
幻樓絢光邊
```
## 鳶尾頌
May 24, 2023 

```
萬紫千紅妖朵朵
騷首婀娜玉肩祼
紗裙嫚舞薄衫彩
鳶尾艷夏風情多
```
