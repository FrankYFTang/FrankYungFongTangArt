## 記初夏後院晨景
Jun 5, 2022

於庚子年端午晨
```
點點紫星碧翠芽
片片綠箔屏垂枝
微曙輕照揮斜影
淡露嫩抺悅閒心
```
## 遙憶嫁紗瀑景
Jun 5, 2022

```
壁堅未敵水柔情
石裂滲破陡礫心
勁風吹薄嫁紗雪
側飄淡掩茫霧形
```

## 优谷鏡湖寫景 
Jun 4, 2022
```
幽谷碧湖影青蒼
```
