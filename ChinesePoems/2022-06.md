## 記初夏後院晨景
Jun 5, 2022

於庚子年端午晨
```
點點紫星碧翠芽
片片綠箔屏垂枝
微曙輕照揮斜影
淡露嫩抺悅閒心
```
## 遙憶嫁紗瀑景
Jun 5, 2022

```
壁堅未敵水柔情
石裂滲破陡礫心
勁風吹薄嫁紗雪
側飄淡掩茫霧形
```

## 道旁焦黑枯根懷情 
Jun 4, 2022
```
焦根枯趄殘林玄
爪枝展憶淚昔怨
路人𢞖𢞖匆匆過
不問烽火離合年
```

## 优谷鏡湖寫景 
Jun 4, 2022
```
幽谷碧湖影青蒼
枝佇柏松蔽靜觴
徐風輕拂貪遊閒
悠步孤林療心傷
```
## 念友寄情花紅 
Jun 4, 2022

2022年六月四日
```
芍藥連牆艷紅媚
牡丹宴歡臥綠叢
歲月靜洗殘瓣淚
嘆學黛玉葬花容
```
