## 油皮囊
Sep 30, 2022
```
不是光頭皆和尚
唯吾禿頂愛發亮
傲留濃眉伴細眼
厚唇粗鼻油皮囊
```

## 嘆近平
Sep 8, 2022
```
胸懷漢武彊土功
欲治康乾盛世平
不行文景仁厚政
終燃紂殷朝歌焚
```

## 弔英女皇
Sep 8, 2022
```
歡宴終需散
王位終需傳
慈君世上少
仁后絕塵寰
```
