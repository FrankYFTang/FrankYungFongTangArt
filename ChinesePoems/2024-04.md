## 苔満林
April 2, 2024 
```
苔藓衣満林
古樹聳高森
巨木臥靜憩
笑我憂凡塵
```

## 烏雲送夕
April 2, 2024 
```
烏雲籠山送夕霞
紫氣難蓋金光華
深湖返映西天彩
獨我一人幽賞暇
```

## 淙淙綠水
April 2, 2024 
```
綠水淙淙流
浪白滾石波
急川時几何
昨暮落雨多
```
