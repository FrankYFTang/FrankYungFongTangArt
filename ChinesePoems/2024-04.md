## 孤鳥沫波
April 16, 2024 
```
孤鳥獨步沫波間
慢巡迴遊自怡憩
雪浪疊層羽白共
渴羨身潔衣淨仙
```

## 苔岩青藓
April 16, 2024 
```
苔岩靜曝艷陽天
綠鮮青藓附石沿
岸浪擊水歌鼓唱
海闊洋遠無境邊
```

## 赤霞閒散
April 15, 2024 
```
赤霞閒散暮色遠
日落光華不再炫
輝煌金碧終西沈
但惜片時夜近緣
```

## 頌舊石橋
April 15, 2024 
```
大石海邊佇
白浪灘沙波
日復千古迴
問人何志多
```

## 黃花環枯
April 11, 2024 
```
黃花點點艷陽下
環開盛茂枯樹旁
千百金瓣蝶未至
春午獨我靜孤賞
```

## 早禱勤
April 9, 2024 
```
天冷艷陽斜暖照
勉我晨起早禱勤
路前長影行似單
忽聞身後問早聲
```

## 苔満林
April 2, 2024 
```
苔藓衣満林
古樹聳高森
巨木臥靜憩
笑我憂凡塵
```

## 烏雲送夕
April 2, 2024 
```
烏雲籠山送夕霞
紫氣難蓋金光華
深湖返映西天彩
獨我一人幽賞暇
```

## 淙淙綠水
April 2, 2024 
```
綠水淙淙流
浪白滾石波
急川時几何
昨暮落雨多
```
