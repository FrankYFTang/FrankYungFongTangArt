## 圖樣圖森破
Dec 3, 2022
```
圖樣圖森破
歐慢一是逮
圖泡圖百歌
敗蓋應拋兒
```

## 樂女心
Dec 2, 2022
```
歌舞昇平醉
滾搖杯酒醺
樂女歡唱後
知誰幕謝心
```

## 宏德來竊
Dec 2, 2022
```
忻言佳語翁
宏志不食禽
德富一身聚
君智兩地飛
```

## 羡李白
Dec 2, 2022
```
獨詩無畫難寫景
杯空酒少勿撈月
莫慕杜甫羡李白
醉思故鄉不憂民
```

## 金葉碎
Dec 2, 2022
```
金葉碎玉落
箔黃舞輕飛
千樹再不綠
靜候年來春
```
