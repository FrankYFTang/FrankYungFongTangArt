## 花香春風
April 24, 2023 

```
花香春風中
嬌陽高空照
百鳥吱嘰唱
莫負時光好
```

## 胭脂緋紅
April 9, 2023 

```
胭脂緋紅滿天霞
林邊水光盡春紗
暗木對問湖中影
風吹波紋可心傷
```
