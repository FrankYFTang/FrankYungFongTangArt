## 庖廚烹
Feburary 13, 2026
```
君子遠庖廚
治國烹小鮮
志懷安國治
何懼孤子君
```
## 南乳蹄膀
Feburary 12, 2026
```
南乳赤煉豬蹄膀
醬浸慢火過夜香
骨離筋糯油走皮
鴿卵豆干熬肉湯
```
燒了三個南乳红燒蹄膀配豆干小鳥蛋.塞了一堆魚肉油麵泡燒大白菜.明晚查經本幫菜伺候
## 惜虹片時
Feburary 11, 2026
```
朝彩虹橋驚現天
對日西東各據天
薄雨笑送天穹盡
厚雲片時摭半天
```

## 春禱
Feburary 11, 2026
```
春雨輕薄寒山霧
朝雲戲掩斜日閒
故夜未飲獨孤酒
晨祈謹禱閤家圓
```

## 待
Feburary 9, 2026
```
添花錦上有何難
送炭雪中情真懷
遊郎初得新車貴
留戀風月几時還
```
