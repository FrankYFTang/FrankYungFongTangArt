## 庖廚烹
Feburary 13, 2026
```
君子遠庖廚
治國烹小鮮
志懷安國治
何懼孤子君
```
## 南乳蹄膀
Feburary 12, 2026
```
南乳赤煉豬蹄膀
醬浸慢火過夜香
骨離筋糯油走皮
鴿卵豆干熬肉湯
```
燒了三個南乳红燒蹄膀配豆干小鳥蛋.塞了一堆魚肉油麵泡燒大白菜.明晚查經本幫菜伺候
## 春禱
Feburary 11, 2026
```
春雨輕薄寒山霧
朝雲戲掩斜日閒
故夜未飲獨孤酒
晨祈謹禱閤家圓
```
